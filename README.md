# Epoch
Epoch is a text-based web game centered around expanding a prosperous civilization. Make homes, forage, and research to develop your society. Made for the Horizons Hack Club Event



## Features
- Grow your population
  - Spend food to grow the population of your civilization
  - Construct houses to support larger populations
- Sustain your population
  - Allocate farmers + make farms to keep your civilization from starving

## Planned Features
- Conquering/Expanding civilization + War
  - defend from raids, take over settlements
- A better way to get research points
- UI Improvements
  - More tooltips, cleaner UI
